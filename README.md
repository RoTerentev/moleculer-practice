# swim-top
This is a [Moleculer](https://moleculer.services/)-based microservices project. Generated with the [Moleculer CLI](https://moleculer.services/docs/0.14/moleculer-cli.html).

## Setup

set gmail credentials in `docker-compose.env`

## Run
```sh
git clone 

docker-compose up --build -d
```

## Description [Russian]
мини-приложение на основе микросервисов, регистрирующее посетителей бассейна и их результат протяженности заплыва с возможностью послать на email письмо сопернику занимающему лидирующую позицию в момент, когда кто-то из посетителей обходит его по результату. Вид приложения представляет из себя шаблон на основе bootstrap(стандартные стили) и React с обновлением блока результатов по websocket(socket.io)

Общий вид приложения являет собой 4 элемента:
 1. Колонка -Список спортсменов с именем/результатом/кнопкой "Обновить результат" и "Добавить спортсмена"
 1. Колонка с общими результатами всех спортсменов, обновляющаяся при изменении
 1. Форма добавления спортсмена в попап-окне
 1. Форма добавления результата открывающаяся в попап-окне при клике на кнопку "Обновить результат"

Обновление результата происходит инкрементально(то есть должно плюсовать к уже имеющемуся значению - введенное)
